Staff Web Development

Job Description

  1. Frontend:
    • Menerjemahkan desain UI/UX kedalam HTML, CSS, JavaScript dan Laravel
    • Mengembangkan dan memelihara komponen-komponen website
    • Mampu membuat pengembangan website yang responsif
    • Berkolaborasi dengan tim Backend untuk memastikan pengalaman pengguna yang optimal.
  2. Backend:
    • Mengembangkan dan memelihara aplikasi web
    • Membangun, mengelola, dan memelihara infrastruktur backend aplikasi, termasuk database, server, dan API
    • Menerapkan langkah-langkah keamanan yang diperlukan untuk melindungi data sensitif
    • Berkolaborasi dengan tim Frontend untuk mengembangkan, dan menguji fitur-fitur baru.
  3. Dibuka posisi baru khusus desain UI/UX

Requirement

  • Frontend: Requirement skill (Basic - Intermediate) : HTML, CSS, Bootstrap, React.JS
  • Backend: Requirement skill (Basic - Intermediate) : PHP, Laravel, MySQL
  • Terbiasa menggunakan GitHub untuk kerja tim
  • Aktif berdiskusi di grup
  • Memiliki cukup waktu luang di sela kesibukan
  • Terbiasa menggunakan software project tracking untuk kerja tim seperti Trello dan sejenisnya
  • Memiliki kemampuan komunikasi efektif dan teamwork yang baik.
  • Memiliki kemauan untuk belajar dan berkembang dalam bidang web development.
  • Diutamakan dari jurusan Teknik Informatika, Ilmu Komputer, Sistem Informasi, atau jurusan terkait lainnya.
